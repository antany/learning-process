CSS Cheatsheet

Adding CSS to HTML

Inline

<div style="background:red">Some content</div>

Using <style> Tag

<head> <style> .sample{ } </style> </head>

.sample is selector, Not Recommended!
Link to CSS file

<link rel='stylesheet' type='text/css' href='css-notes.css'>

Recommended Way .content of external .css file and style tag will follow the same syntax

Importing External Fonts

Import Fonts - CSS

@import url('https://fonts.googleapis.com/css2?family=Antonio:wght@100&display=swap');

can be used in CSS like "font-family: 'Antonio',sans-serif;" sans-serif - default to browser default sans-serif font
Import Fonts - Header Link Tag

<link rel="preconnect" href="https://fonts.googleapis.com"> <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin> <link href="https://fonts.googleapis.com/css2?family=Antonio:wght@100&display=swap" rel="stylesheet">

can be used in CSS like "font-family: 'Antonio',sans-serif;" sans-serif - default to browser default sans-serif font

Selectors

Element Selector

h1 { //SomeStyle }

Style applies to all h1 html elements
Class Selector

.nodes { //SomeStyle }

Style applies to elements, where class="nodes"
Universal Selector

*{ //Some Style }

Rarely use this one, applies to all elements
ID Selector

#idname{ //Some Style }

Style applies to elements, where id="idname"
Attributes Selector

[disabled]{ //Some Style }

Style applies to elements, which are disabled
